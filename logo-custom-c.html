Recommendation

70% Width (Option 3) gives the best balance — the C feels tighter/more secure without looking squashed. The narrower aperture subconsciously signals "closed loop" and "containment" which aligns with the firewall metaphor.

Custom SVG (Option 4) if you want perfect control, but requires maintaining the SVG file. The scaleX approach (Option 3) is easier to implement in CSS/HTML.
