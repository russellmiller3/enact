Receipt Detail

Connect to Enact Cloud

Paste the API key from your Enact Cloud account to view your team's audit receipts. You can find it at Settings → API Keys in the Enact dashboard.

Your key stays in this browser's local storage and is only sent to your Enact Cloud instance. All receipt data is zero-knowledge encrypted and append-only — Enact cannot read your payloads.
